int * p = &a; Declare a pointer-type of variable name p, and initialize to the address of variable a. In a declaration, "*" stands for "pointer-type". Afterwards, "*" means"de-reference", retrieving the value at the memory location, using the syntax *p. 

int main() { int a = 0; int *p = &a; printf("%ld", ((long)(p))); return 0; } 140722421153132

*(p+66) = p[66] stands for starting at offset 66 position,read the value at the new location
